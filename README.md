# bs-testimonials
Testimonial Plugin

# ShortCode
[bstestimonial slider=true]

# Options
count {eg. count=3} || default: 1 <br/>
authour {eg. authour=true/false} || default: false <br/>
title {eg. title="define title here"} || default: false <br/>
slider {eg. slider=true/false} || default: false <br/>
